bePWR SCORE

Tu Indice de Vitalidad Funcional — calificacion de 0 a 100 que refleja tu estado fisico actual basado en metricas objetivas.

COMO SE CONSTRUYE
📊 Calificacion InBody — 35%
🔥 % Grasa corporal vs rangos saludables — 25%
💧 Grasa visceral — 20%
💪 Fuerza funcional (mancuernas / peso) — 10%
📏 Cintura vs referencia OMS — 10%
COMO SE INTERPRETA
● 0–55 — En Proceso
● 56–72 — Optimo
● 73–87 — Avanzado
● 88–100 — Destacado
Se actualiza con cada nueva valoración.
TENDENCIA

Compara tus métricas actuales contra tu valoración anterior y calcula un puntaje de cambio real.

CÓMO SE CALCULA
💪 Masa muscular — +4 si mejora
💧 Grasa visceral — +3 si baja
📊 Calificación InBody — +3 si sube
🔥 % Grasa corporal — +2 si baja
⚖ Peso (según objetivo) — +2
📏 Brazo / Pantorrilla — +2 cada una
🏋 Mancuernas — +2 si mejora
📍 Cintura / Abdomen — +1 cada una
CÓMO SE INTERPRETA
● +4 o más — Progreso
● -3 a +3 — En Proceso
● -4 o menos — Retroceso
Requiere al menos 2 valoraciones.
